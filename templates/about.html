About Indelible.org

Indelible.org is maintained by Jon Parise as a place to publish notes, articles, and projects.

Biography

Jon Parise is senior software engineer at Electronic Arts. He has worked on a number of titles, including The Sims Online and Ultima Online, and is currently working on The Sims 3.

He has been active in a number of open source projects, including PHP, PEAR, and The Horde Project. He also thinks that Python is the bee's knees.

Selected print publications include:

  • Beginning Databases with PostgreSQL — technical reviewer
  • Professional PHP4 Programming — contributing author
  • Massively Multiplayer Game Development 2 — contributing author

Jon earned a bachelors degree in Information Technology from the Rochester Institute of Technology and a masters degree in Entertainment Technology from Carnegie Mellon University.

Colophon

This site is built upon a stack of open source technologies:

  • The Apache Web Server with mod_wsgi
  • The Python Programming Language
  • The Vim text editor
  • The Mercurial distributed version control system
  • PostgreSQL: The world's most advanced open source database
  • Django: The Web framework for perfectionists with deadlines
  • Django Ink: A simple blog-like publishing application for Django
  • Docutils: Text processing and rendering using reStructuredText
  • Pygments: A generic syntax highlighter
